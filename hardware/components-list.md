# Hardware Components List

Daftar komponen yang digunakan pada proyek robot COLOR FLAG LIFTING MISSION.

---

## Main Components

| Component | Quantity | Description |
|-----------|----------|-------------|
| Arduino Uno / Nano | 1 | Mikrokontroler utama |
| HC-05 Bluetooth Module | 1 | Komunikasi Bluetooth |
| Servo Motor | 1 | Mekanisme pengangkat bendera |
| DC Gear Motor | 2 | Penggerak roda robot |
| Motor Driver | 1 | Pengontrol motor DC |
| Color Sensor | 1 | Pembaca warna arena |
| Robot Chassis | 1 | Struktur utama robot |
| Wheels | 2 | Roda robot |
| Li-ion Battery | 1 | Sumber daya utama |

---

## Supporting Materials

- Jumper cables
- Breadboard
- Switch button
- Acrylic / PVC frame
- Screws and mounting hardware

---

## Functional Overview

### Arduino
Mengontrol seluruh sistem robot termasuk pembacaan sensor, komunikasi Bluetooth, dan kontrol servo.

### HC-05 Bluetooth Module
Digunakan untuk menerima perintah gerak dari smartphone operator.

### Color Sensor
Mendeteksi warna arena untuk menentukan bendera yang harus diangkat.

### Servo Motor
Mengangkat dan menurunkan bendera sesuai misi.

### Motor Driver & DC Motor
Mengatur pergerakan robot ke berbagai posisi dalam arena pertandingan.

---

## Power System

Robot menggunakan baterai Li-ion sebagai sumber daya utama untuk:
- Mikrokontroler
- Motor DC
- Servo
- Modul Bluetooth
